# Quasort Up

 Quasort Up - Умное веб-приложение способное помочь вам с тренировками! Главной особенностью является "Блокнот" c возможностью глубокого анализа данных на основе вашего прогресса в течение некоторого времени.
 

## Функционал

- Интерактивная модель человека ***Rock*** с информацией обо всех мышцах
- Умный блокнот
- Счетчик КБЖУ
