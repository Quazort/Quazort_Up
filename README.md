# Quazort_Up

Quazort_Up — приложение, которое значительно облегчает вашу спортивную жизнь. В нем вы можете вести свои тренировки,
записывать результаты и анализировать прогресс с помощью ИИ-ассистента.

Если у вас нет опыта тренировок, вам пригодится Rock — наглядная схема человека с выделенными мышцами. Достаточно нажать
на любую мышцу, чтобы получить всю необходимую информацию о ней.

Кроме того, приложение содержит множество качественных советов от лучших спортсменов со всего мира.
